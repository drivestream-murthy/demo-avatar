# Avatar Gen — Portrait ERP (Simple, Node 20.x, no vercel.json)

- Static HTML+JS with CDN HeyGen SDK
- Node 20.x via package.json
- API routes in /api
- Portrait stage, video dock, voice + typed input, background auto change

## Deploy
1) Import to Vercel
2) Add env var HEYGEN_API_KEY
3) Deploy, test /api/token
4) Open site → Start → Unmute if needed
